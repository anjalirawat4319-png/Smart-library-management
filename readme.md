# Smart Library Management System

Python OOP Final Project

## Project Overview
This project demonstrates core Object-Oriented Programming concepts:
- Encapsulation
- Inheritance
- Polymorphism
- Composition
- File Persistence (JSON)

## Features
- Add books
- Add members
- Borrow books
- Return books
- Save data to JSON files
